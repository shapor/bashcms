Source Code

Overview

The user-space FUSE implementation is available in the Mercurial repository hosted at hg.tux.org linked below. It shares code with the kernel port, which is making rapid progress.

For getting the Tux3 kernel code up and running in UML, see: http://mailman.tux3.org/pipermail/tux3/2008-November/000380.html

Source Repositories

  • Official repository - official Mercurial repository/code browser. Clone with hg clone http://hg.tux3.org/tux3/
  • Tux3 kernel git repository - official kernel git tree hosted at kernel.org. Clone URL: git://git.kernel.org/pub/scm/linux/kernel/git/daniel/linux-tux3.git
  • Tux3 at Bitbucket - bitbucket.org mirror of the official repository (nicer web interface) RSS feed
